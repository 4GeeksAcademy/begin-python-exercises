 # `106` Elementos pares

## 📝 Instrucciones:

Dado una lista de números, imprime todos sus elementos pares. Usa un loop o bucle for para iterar la lista en sí y no sus índices. 
¡No uses range()!

**Ejemplo de entrada**
1 2 2 3 3 3 4

**Ejemplo de salida**
2 2 4
