 # `86` Eliminar un caracter

## 📝 Instrucciones:

Dado un string, elimina todos los caracteres @ de dicho string.

**Ejemplo de entrada**
Bilbo.Baggins@bagend.hobbiton.shire.me

**Example output**
Bilbo.Bagginsbagend.hobbiton.shire.me
