# `035` net amount

1. Write a program that computes the net amount of a bank account based a transaction log from console input. The transaction log format is shown as following:

D 100
W 200

`D` means deposit while `W` means withdrawal.

## Example input:

+ D 300
  D 300
  W 200
  D 100

## Example output: 

+ 500
