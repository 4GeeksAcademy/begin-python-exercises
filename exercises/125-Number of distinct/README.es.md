 # `125` Cantidad de números distintos

## 📝 Instrucciones:

Dado una lista de números enteros, cuenta cuantos números distintos tiene. Esta tarea se puede resolver en tan solo una línea.

**Ejemplo de entrada**
1 2 3 2 1

**Ejemplo de salida**
3
