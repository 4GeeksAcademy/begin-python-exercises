 # `110` Cantidad de elementos distintos

## 📝 Instrucciones:

Dada una lista de números con todos sus elementos en orden ascendente, determina e imprime la cantidad de elementos distintos en ella.

**Ejemplo de entrada**
1 2 2 3 3 3

**Ejemplo de salida**
3
