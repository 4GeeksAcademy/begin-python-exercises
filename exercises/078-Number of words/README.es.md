 # `78` Cantidad de palabras

## 📝 Instrucciones:

Dado un string que consiste en palabras separadas por espacios, determina cuantas palabras tiene. Para resolver este ejercicio usa el método count.

**Ejemplo de entrada**
Hello world

**Ejemplo de salida**
2
