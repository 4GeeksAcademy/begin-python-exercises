 # `126`- Números en común

## 📝 Instrucciones:

Dado dos listas de números, cuenta cuántos números de la primera lista se repiten en la segunda. 

**Ejemplo de entrada**
1 3 2
4 3 2

**Ejemplo de salida**
2
